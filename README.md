# MultiConvert

A currency and crypto converter for iOS 17+. Shows your 10 most recently used currencies converted simultaneously from a single base amount — no tab switching, no menus.

## Features

- **Live multi-currency display** — 10 recent currencies updated in real time
- **MRU (most recently used) cache** — pick a currency, it jumps to position 1
- **Fiat + Crypto** — USD, EUR, JPY, MXN, GBP, CAD, AUD, CHF, CNY, KRW, BRL + BTC, ETH, SOL, USDC, USDT
- **Custom dark keypad** — inspired by (not copied from) Apple Calculator
- **Offline mode** — cached rates with staleness indicator (>24h)
- **Pull to refresh** — tap the staleness bar or pull the list
- **Home Screen Widget** — small (2 pairs) and medium (4 pairs)
- **Settings** — decimal places (2 or 4), widget base amount/currency, cache control

## Setup

### Requirements

- Xcode 15.0+
- Swift 5.10+
- iOS 17.0+ deployment target
- No external dependencies (pure Apple frameworks)

### Running

```bash
git clone <this-repo>
cd MultiConvert

# Regenerate the project file if needed (after adding source files)
python3 gen_xcodeproj.py

# Open in Xcode
open MultiConvert.xcodeproj
```

Select the **MultiConvert** scheme, choose an iOS 17+ simulator, and press Run.

### Running Tests

```bash
xcodebuild test \
  -project MultiConvert.xcodeproj \
  -scheme MultiConvert \
  -destination 'platform=iOS Simulator,name=iPhone 16' \
  -only-testing:MultiConvertTests \
  CODE_SIGNING_ALLOWED=NO
```

Expected: **42 tests, 0 failures**

### Adding Source Files

The Xcode project is generated by `gen_xcodeproj.py`. To add a new file:

1. Create the `.swift` file in the appropriate subdirectory
2. Add its path to the correct source list in `gen_xcodeproj.py`
3. Run `python3 gen_xcodeproj.py` to regenerate `project.pbxproj`

## API Rate Limits

| API | Used For | Rate Limit | Key Required |
|-----|----------|-----------|--------------|
| [frankfurter.app](https://www.frankfurter.app) | Fiat exchange rates | ~1 req/s (generous) | No |
| [CoinGecko free tier](https://www.coingecko.com/api) | Crypto prices | 10–30 req/min | No |

**Note:** The spec listed `api.exchangerate.host` for fiat rates, but that service now requires a paid subscription. `frankfurter.app` is used instead — it's open-source, free, and returns the same format. See `DECISIONS.md` for full rationale.

Rates are cached to disk and reused for 24 hours to minimize API calls.

## Conversion Formula

All rates are stored internally as **units per USD**:

```
result = amount × ratesPerUSD[targetCurrency] / ratesPerUSD[baseCurrency]
```

Example: 100 EUR → BTC (BTC price = $65,000)
```
ratesPerUSD["EUR"] = 0.926        (1 USD = 0.926 EUR, so 1 EUR = 1.08 USD)
ratesPerUSD["BTC"] = 1/65000      (1 USD = 0.0000154 BTC)

result = 100 × (1/65000) / 0.926 = 0.001661 BTC
```
